各大编程语言的HelloWorld

'Hello,World!' 中文意思是“你好， 世界”。

因为 《The C Programming Language》 中使用它做为第一个演示程序。

后来的程序员在学习编程或进行设备调试时延续了这一习惯。

——百度百科

Python

print("Hello, World!")

C

#include<stdio.h>
int main() {
    printf("Hello, World!")
    return 0;
}

C++

#include<iostream>
using namepsace std;
int main() {
    cout << "Hello, World!";
    return 0;
}

C#

using System;

internal static class HelloWorld {
  private static void Main() {
    Console.WriteLine("Hello, World!");
  }
}

Objective-C

#import <Foundation/Foundation.h>

int main() {
    @autoreleasepool {
        NSLog(@"Hello, World!");
    }
}

JavaScript

document.write('Hello, World!');

Java

class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Ruby

puts "Hello, World!"

Rust

fn main() {
    println("Hello, World!");
}

Go

package main

import "fmt"

func main() {
  fmt.Println("Hello, World!")
}

Scale

object HelloWorld extends App {
  println("Hello, World!")
}

Lua

print("Hello, World!")

PHP

<? echo "Hello, World!" ?>

swift

print("Hello, World!")

Prel

print "Hello, World!\n";

汇编

bdos    equ    0005H    ; BDOS entry point

        lxi    d,msg$   ; address of msg
        call   bdos
